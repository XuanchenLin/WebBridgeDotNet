## WebBridge.NET

### 🚀 关于项目

这是一个开源的 .NET 库，旨在弥合 Web 应用程序与本机操作系统之间的鸿沟。你可以在浏览器的 JavaScript 上下文中公开自定义 API，允许网页直接异步地与本地系统资源进行交互。

现代 Web 应用程序被限制在浏览器的沙盒环境中。WebBridge 通过在浏览器和本地运行的 .NET 后端之间建立通信通道，为这些限制提供了一个受控的突破途径。通过将你的 .NET 代码绑定到 JavaScript 环境，你可以赋予 Web 应用程序利用底层操作系统全部能力的功能。

### ✨ 核心特性

- **调用本机 API：** 直接从 JavaScript 函数执行本地 .NET 代码
- **启动和管理应用程序：** 从 Web UI 启动、监控和交互桌面应用程序
- **与外设交互：** 从扫描仪、传感器或其他自定义硬件读取数据
- **全栈集成：** 无缝集成丰富的 Web 前端与强大的 .NET 后端逻辑

### 🖥️ 系统要求

**开发环境：**

- Visual Studio 2022
- .NET 8.0 或更高版本

**支持平台：**

- Windows / macOS / Linux
- 现代 Web 浏览器（Chrome、Firefox、Edge、Safari）
